# Solar Power Forecasting

Predicting solar power generation using machine learning.

## Overview

This project predicts solar power output based on weather and environmental conditions. Accurate forecasting helps grid operators manage electricity supply and demand.

## Dataset

- Source: [Kaggle - Solar Power Generation Data](https://www.kaggle.com/datasets/anikannal/solar-power-generation-data)
- Features: Irradiance, temperature, time of day, weather conditions
- Target: Power output (kW)

## Tech Stack

- Python (pandas, numpy)
- Scikit-learn, XGBoost
- Matplotlib, Seaborn
